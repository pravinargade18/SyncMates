# SyncMates

# Website - <em>[SyncMates](https://sync-mates.vercel.app/)</em>


### How to Run Locally
Run the below command to clone the repo locally
```
git clone https://github.com/pravinargade18/SyncMates.git
``` 

### To run the application, run below commands 
```
npm install
npm run dev
```
